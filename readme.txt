如何解决（提高保存图片的质量和分辨率）：

最直接的方法是提高 html2canvas 的 scale 选项值。

增加 scale 值： 将 scale: 2 更改为 scale: 3、scale: 4 甚至更高。更高的 scale 值会指示 html2canvas 生成更大的画布和更多的像素，从而更接近原始图片的质量和分辨率。 例如，如果将 scale 设置为 4，那么一个在屏幕上显示为 500 像素宽的图片，将被捕获为 2000 像素宽。
权衡和注意事项：

内存和性能消耗： 显著提高 scale 值会大幅增加 html2canvas 运行时的内存消耗和处理时间。对于非常大的图片文件，设置过高的 scale 值可能导致：
浏览器变慢，界面卡顿。
浏览器崩溃（特别是内存不足时）。
下载的 Data URL 过长，某些浏览器或下载管理器可能处理不佳。
找到平衡点： 您需要根据实际输入的图片大小和您期望的输出质量，以及您的浏览器性能，来尝试不同的 scale 值，找到一个最佳的平衡点。
我为您将代码中的 scale 值从 2 调整为 3。您可以尝试运行此版本，然后根据需要进一步调整 scale 值（例如改为 4）。
